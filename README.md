# crane

Early GitHub sync preview of the `crane` package.

This repository is being split out from a larger local research workspace. The
current goal is to keep a clean, incremental GitHub history around the `crane`
package itself, not to present a finished public release.

## Current scope

- package code under `crane/`
- minimal packaging via `pyproject.toml`
- a small example in `examples/quickstart.py`

## Current non-goals

- complete CLI workflow
- full documentation set
- bundled datasets or experiment outputs
- legacy `couture` workspace material

## Install

```bash
pip install -e .
```

## Quick start

See `examples/quickstart.py`.

## Status

This is an early preview snapshot. Interfaces may still change while the
repository boundary and packaging are being cleaned up.
